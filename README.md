## Arrow Keys

zsh need to unbind arrow related keys.

```zsh
## unbind keys
bindkey -r "^[OA"
bindkey -r "^[OB"
bindkey -r "^[OC"
bindkey -r "^[OD"
bindkey -r "^[[1;5A"
bindkey -r "^[[1;5B"
bindkey -r "^[[1;5C"
bindkey -r "^[[1;5D"
bindkey -r "^[[A"
bindkey -r "^[[B"
bindkey -r "^[[C"
bindkey -r "^[[D"
```
