### Requirements

* GNU sed (macos)
  ```zsh
  brew install gnu-sed
  ```

* fd: https://github.com/sharkdp/fd?tab=readme-ov-file#installation
  - macos
    ```zsh
    brew install fd
    ```

* rg: https://github.com/BurntSushi/ripgrep?tab=readme-ov-file#installation
  - macos
    ```zsh
    $ brew install ripgrep
    ```
